Introduction

Welcome to the Competitive Price Intelligence documentation. This comprehensive suite of machine learning packages is designed to empower retailers and market analysts with cutting-edge tools for competitive analysis, price optimization, and market intelligence.

Our packages leverage state-of-the-art unsupervised learning algorithms, advanced time series forecasting models, and sophisticated statistical analysis techniques to extract actionable insights from complex pricing data ecosystems.

Industry-Grade Solutions: These packages are built to handle real-world retail challenges with production-ready implementations, comprehensive error handling, and scalable architectures.

Getting Started

Follow these steps to begin using our ML packages in your retail analytics workflow:

1. Installation

Install packages directly from our GitHub repository using pip:

# Install Competitor Identification package
pip install git+https://github.com/isrkan/Competitive_Price_Intelligence.git#subdirectory=ml_packages/competitor_identification

# Install Price Imputation and Forecasting package
pip install git+https://github.com/isrkan/Competitive_Price_Intelligence.git#subdirectory=ml_packages/price_imputation_and_forecasting

2. Quick Start Example

Here's a simple example to get you started with competitor identification:

from competitor_identification import CompetitorIdentification
from competitor_identification.pipeline import run_pipeline

# Initialize
ci = CompetitorIdentification()
config = ci.get_config()

# Run complete pipeline
results = run_pipeline(
    config=config,
    data_directory_path='./data',
    category='rice',
    SubChainName='Tesco Express',
    StoreName='Tesco Express - Oxford Street',
    product_description='Basmati Rice 1kg',
    Geographic='City',
    include_clustering_evaluation=True
)

# Access results
top_competitors, fig1, fig2, fig3, silhouette, davies_bouldin = results
print(f"Identified {len(top_competitors)} competitors")
print(f"Cluster quality score: {silhouette:.3f}")
